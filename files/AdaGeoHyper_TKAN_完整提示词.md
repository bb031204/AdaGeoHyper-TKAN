## 空间超图构建要求（必须严格保留）

本模型的空间部分采用一个基于 **经度、纬度、海拔** 的自适应邻域超图来刻画多站点之间具有物理意义的高阶空间关系。  
请严格按照以下原理实现，不得偏离我的思路：

### 空间建模总体思想

* 将空间建模与时序建模明确解耦；
* 空间部分采用一个基于经度、纬度和海拔的自适应邻域超图来刻画多站点之间具有物理意义的高阶空间关系；
* 时序部分由 TKAN 负责，对输入的 12 步历史序列进行统一建模，并直接输出未来 12 步结果，从而实现 12→12 的非自回归多步预测。

### 节点定义

设共有 `N` 个站点，第 `i` 个站点对应一个节点 `v\\\_i`，其静态空间属性记为：

```text
p\\\_i = \\\[lon\\\_i, lat\\\_i, alt\\\_i]
```

### 距离定义

首先利用经纬度与海拔构造站点之间的三维地理距离，以保证超图的空间连接具有明确的气象物理意义。  
两个站点 `v\\\_i` 和 `v\\\_j` 的距离定义为：

```text
d\\\_ij = sqrt( λ\\\_g \\\* d\\\_geo(i,j)^2 + λ\\\_h \\\* (alt\\\_i - alt\\\_j)^2 )
```

其中：

* `d\\\_geo(i,j)`：由经纬度计算得到的球面距离；
* `λ\\\_g`、`λ\\\_h`：分别控制平面距离与海拔差在总距离中的权重。

### 超边构造

基于上述距离，对每个站点执行 **K 近邻搜索**，得到其地理候选邻域 `N\\\_i^geo`，进一步构造邻域超边：

```text
e\\\_i = {v\\\_i} ∪ N\\\_i^geo
```

也就是说，每个站点都对应一条以自身为中心、连接其局部地理邻居的超边，从而形成一个以地理位置为骨架的邻域超图。

### 关联矩阵

若记关联矩阵为 `H`，则：

* 当节点 `v\\\_i` 属于超边 `e\\\_j` 时，`H\\\_ij = 1`
* 否则为 `0`

### 超图建模意义

与普通图中的两两连接不同，一条超边可以同时连接多个站点，因此更适合表达局部区域内多个站点共同演化的高阶空间依赖。

### 自适应权重机制

为了避免仅使用固定地理邻域带来的过强静态假设，模型进一步引入一种轻量自适应机制，用来根据当前输入时间窗口内的站点状态对邻域权重进行动态调整。

要求：

* 不重新改变超边成员；
* 保持“地理邻域骨架固定、邻域贡献强度动态变化”的设计；
* 既保留空间结构的稳定性，又能反映不同天气过程下局地关系强弱的变化。

### 状态摘要

设第 `i` 个站点在当前输入窗口中的观测序列为：

```text
X\\\_i = \\\[x\\\_i^(1), x\\\_i^(2), ..., x\\\_i^(12)] ∈ R^(12×F)
```

先对该时间窗口进行轻量汇聚，得到站点状态摘要：

```text
s\\\_i = Pool(X\\\_i)
```

这里的 `Pool(.)` 可以采用：

* 平均池化；
* 最后时刻特征提取；
* 一层小型线性映射后的低维表示。

其目的不是替代 TKAN 的时序建模，而只是提供一个简洁的当前状态描述。

### 局部打分函数

对于中心站点 `v\\\_i` 及其候选邻居 `v\\\_j ∈ {v\\\_i} ∪ N\\\_i^geo`，构造轻量打分函数：

```text
u\\\_ij = φ(\\\[p\\\_i, p\\\_j, s\\\_i, s\\\_j])
```

其中 `φ(.)` 可以是：

* 两层 MLP；
* 或线性打分器。

通过 softmax 在局部邻域内归一化，得到自适应邻域权重：

```text
α\\\_ij = exp(u\\\_ij) / Σ\\\_k exp(u\\\_ik)
```

### 空间传播

在空间传播阶段，模型基于上述带权超图对站点特征进行高阶聚合。  
对于第 `i` 个站点，其空间增强表示为：

```text
z\\\_i = Σ\\\_{j∈e\\\_i} α\\\_ij x\\\_j W
```

其中：

* `x\\\_j` 表示节点 `v\\\_j` 的输入特征或隐藏表示；
* `W` 为可学习映射矩阵。

### 标准超图卷积矩阵形式（可选实现表达）

若采用矩阵化的标准超图卷积形式，也可写为：

```text
Z = σ(D\\\_v^(-1/2) H W\\\_e D\\\_e^(-1) H^T D\\\_v^(-1/2) X W)
```

其中：

* `D\\\_v` 为节点度矩阵；
* `D\\\_e` 为超边度矩阵；
* `W\\\_e` 可结合上述自适应邻域权重来体现动态空间依赖。
